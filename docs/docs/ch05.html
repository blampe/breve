Chapter 5. What's New in Version 2.5.1

breve version 2.5.1 introduces several new features and changes.

New Features in Version 2.5.1

breve 2.5.1 is a bug-fix release that introduces one major new feature (currently experimental):

  • Experimental new Chapter 18 allows Python code to be called from breve simulations

Below are some of the major new features found in breve 2.5 . For a full list of new features, see the "Changes" file that is included with the breve distribution.

  • Network connections may now return objects to the client which initiated the connection [Chapter 17]

  • Updated plugin and language frontend API [the section called “Sending and Receiving breve Objects”]

  • Major bug fixes and stability improvements
